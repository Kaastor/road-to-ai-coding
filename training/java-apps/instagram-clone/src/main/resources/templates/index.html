Instagram Clone

Welcome to the Instagram Clone web interface!

Available Features:

  • User Registration: Create a new account with username, email, and password
  • User Login: Authenticate with username/email and password
  • Create Posts: Share content with other users
  • View Posts: Browse posts from specific users

Quick Start:

  1. Register a new account
  2. Login with your credentials
  3. Create and view posts
